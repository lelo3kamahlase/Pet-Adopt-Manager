Pet Adoption Management System
A C# ASP.NET web application designed to manage animal shelter operations, including pet tracking, adoption records, and donations.

Features
Pet Management: Track animal details and adoption status.
Donation System: Record and manage donor contributions.
Adoption Tracking: Manage records linking pets to their new owners.

Tech Stack
Language: C#
Backend: ASP.NET with Entity Framework
Database: SQL Server (Relational)
